container elements

<span> and <div> are both container elements:
they don't have their own content, but only contain content, still they can have styles attached to them

inline and block tags

What is also interesting here, is the difference between inline and block tags in HTML

inline tags continue the line as it was

<b>     <span>     <img>     <br>     <a>
while block tags create a box around the content enwrapped by them.
<p>     <div>
It's sweet to know this for formatting.
