I have the feeling that it hardly makes sense to simply copy someone else's code that I wouldn't understand. At least I don't like this much, because I don't have the feeling I'll learn much from it.

But programming is somehow incremental: First, coming up with the general idea. Then building a bit here and then a bit there. And with lots of those little building blocks, finally the program starts coming together. This is in my opinion also the reason why Object oriented programming is a very useful way of programming, because with Classes and objects, it creates such an environment of many self-sufficient "blocks". If I have bits of code that work and I know what I can use them for, then I can re-use them and apply them also in a different project, without having to write everything anew from scratch, thereby making the workflow much faster and more efficient. The use of Classes can also contribute to an increased security, since variables and methods described in the Class will only be made available to instances of that same Class, therefore avoiding complications with other parts of the program.

OOP also offers the chance to write a clear and clean modular structure, that makes it easy to maintain and modify existing code (be it that you wrote it yourself or someone else did). It is easy to create new instances, and even to create new objects through inheritance, which might require only a small amount of work to create, while still being highly functional. To refer here to my example Class on the following page: if there is a parent Class "Travels" that defines methods to calculate duration and distance, these methods could easily be inherited to possible children Classes, such as "Biking" or "Hiking". This is nicely logical, clean, and saves a lot of work.

Also other people's code can often be one of these building blocks. And maybe their code doesn't fit exactly, but the cool thing about these blocks is, that they are completely moldable. I can take, change a bit on that corner and make it so the pieces fit together into my own puzzle.
